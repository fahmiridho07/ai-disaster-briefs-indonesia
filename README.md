# ai-disaster-briefs-indonesia

> 🚧 **Work in progress** — this project is currently on hold and will be continued. Scope and notebooks are still being developed.

Data analytics & IBM Granite-generated summaries for Indonesia's disaster risk, based on BNPB / IRBI data (2019–2024).

## Goal

Combine exploratory data analysis of Indonesia's disaster risk indicators with concise, AI-generated briefs (powered by IBM Granite) to make the findings easier to read and act on.

## Planned Scope

- Clean and explore BNPB / IRBI disaster risk data (2019–2024)
- Summarize key risk insights per region
- Generate natural-language briefs using IBM Granite

## Status

Early-stage / paused. More to come — stay tuned.
